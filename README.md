# 🏠 БУНКЕР - Онлайн

Постапокалиптическая настольная игра о выживании.

## 🎮 Как играть

### Вариант 1: Локально (без сервера)
```bash
npm install
npm run dev
```
Игра откроется по адресу http://localhost:3000

### Вариант 2: С сервером (полный онлайн)
```bash
# Установить зависимости
npm install

# Запустить сервер (в одном терминале)
npm run server

# Запустить клиент (в другом терминале)
npm run dev
```

## 🎯 Особенности

- **100+ карт** - профессии, здоровье, хобби, фобии, предметы
- **Маньяк** - скрытая роль в режиме "Безумие"
- **Разведка** - отправляйся на поиски припасов
- **События** - джин, радиация, маньяк
- **Система очков** - скрытый подсчёт результатов

## 🛠️ Технологии

- Frontend: React + TypeScript + TailwindCSS
- Backend: Node.js + Socket.io
- Стиль: Постапокалиптический тёмный

## 📝 Правила

1. Создай комнату или присоединись по коду
2. Получай карты с характеристиками
3. Решай угрозы, используя предметы
4. Выживи или стань последним!
